#### Evaluating Housing Accessibility and Spatial Disparities in Shanghai
BUSS2510 - Data Collection and Visualization 

## Project Overview
This project evaluates housing affordability across Shanghai's 16 districts through comprehensive web scraping, data analysis, and a relational database implementation. By collecting current housing listings, rental data, historical price trends, and income statistics, we analyze spatial price disparities, rental patterns, and affordability metrics including price-to-income and price-to-rent ratios.

## Key Research Questions
- How do housing prices vary across Shanghai districts?
- What are the rental market patterns by district?
- How affordable is housing relative to resident income?
- What factors drive housing price variations?

## Disclaimer

All code within this project was originally developed in Jupyter Notebooks (`.ipynb`) and subsequently converted to Python scripts (`.py`) using `nbconvert` for production deployment and version control. The original notebooks are preserved in the `notebooks/` directory and should be referenced as the primary source for development history, exploratory analysis, and iterative code refinement.
